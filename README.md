# 🅿️ ParkSmart — Intelligent Smart Parking Dashboard

An elegant, single-page intelligent parking simulation and event reservation dashboard built using pure vanilla web technologies. Features responsive glassmorphic interfaces, dynamic real-time slot selection grid maps, ticket vector QR rendering, and a state management lifecycle.

## 🚀 Key Architectural Pillars

* **Vanilla Web Stack:** Formatted using native HTML5, modular JavaScript functional architectures, and procedural CSS3 parameters[cite: 2].
* **Persistent Client State Engine:** Leverages browser `localStorage` hooks to maintain a read-only simulated relational table structure across page refreshing intervals[cite: 2].
* **Deterministic Vectors:** Implements an algorithm to dynamically generate procedural vector QR codes inside digital transaction tickets on-the-fly[cite: 2].
