# 🅿️ ParkSmart — Intelligent Parking Simulation Dashboard

An elegant, high-performance single-page intelligent parking system and event reservation dashboard built using pure vanilla web technologies. Features a responsive, hyper-modern glassmorphic interface, real-time dynamic slot-allocation maps, and persistent state management without external server dependencies.
